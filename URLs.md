# Available Services and URLs

## Frontend

-   **URL:** [http://localhost:3000](http://localhost:3000)
-   **Description:** This is the main web application interface.

## API

-   **URL:** [http://localhost:5050](http://localhost:5050)
-   **Description:** Backend API service for handling requests and database interactions.

## MongoDB Database

-   **Service Name:** `mongo`
-   **Internal Only:** Not directly accessible via a browser.
-   **Admin Username:** `root`
-   **Admin Password:** `very-strong-db-password`

## Mongo Express (MongoDB Admin Panel)

-   **URL:** [http://localhost:8081](http://localhost:8081)
-   **Description:** Web-based admin interface for MongoDB.

## Notes

-   MongoDB is only accessible internally and should not be exposed directly.
-   Mongo Express provides a UI to manage the database.
